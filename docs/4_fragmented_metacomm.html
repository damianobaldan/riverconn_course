Fragmentation and metacommunities

  • Metacommunity: a set of local communities linked by dispersal
  • Local communities are influenced by both local environmental conditions and regional dispersal processes
  • Mass effect (ME) archetype: high dispersal -> ‘everything is everywhere’
  • Species sorting (SS) archetype: low dispersal -> high effect of local environment (filter)
  • Dispersal (connectivity) strength regulates between ME and SS

Fragmentation and metacommunities

  • In a landscape, central sites are more connected and peripherical sites are more isolated
  • We expect effect of centrality/isolation on alpha and beta diversity
  • River network have per se strong spatial structure
  • dams affect this spatial structure
  • riverconn indices can help quantifying this ‘new’ spatial structure

Which packages we will use

  • dplyr for data manipulation: helps building pipelines for easy-to-read data manipulation (loaded with the tidyverse)
  • ggplot2 for data visualization: a bit painful, but easily integrates sf and terra objects for plotting (loaded with the tidyverse)
  • vegan for ecological analyses: provides functions for calculating diversity indices and performing Mantel tests
  • MASS for the negative binomial GLM
  • adespatial for calculating beta diversity
  • ecodist for the Mantel test with multiple predictors
  • betareg and StepBeta for Generalized Linear Models with beta link function

The Ebro case study: beta diversity

The Ebro case study: beta diversity

Next we want to analyze what are the drivers of beta diversity. First we calculate the beta diversity matrix with adespatial::beta.div() function. We use the Sorensen indices, and we are interested in three components of the beta diversity: the total beta diversity (D), the richness difference (rich), and the replacement (repl), as they capture different community asssembly processes.

The Ebro case study: beta diversity

We use a Mantel test (ecodist::mantel()) to check the influence of fragmentation on beta diversity. First we calculate distances between predictors. We also retain the lower triangle of the distance matrix for later use.

dist_data <- cbind(
  "AGR" = ecodist::distance(env_mat$AGR  %>% car::logit() %>% scale %>% as.vector),
  "UR" = ecodist::distance(env_mat$UR  %>% car::logit() %>% scale %>% as.vector),
  "us0.3" = ecodist::distance(frag_mat$index_us0.3 %>% scale %>% as.vector),

The Ebro case study: Local Contribution to Beta Diversity (LCBD)

The Ebro case study: LCBD

Finally, we analyze LCBD, that is the contribution of each site to the total beta diversity. We check if LCBD is related to fragmentation. We first calculate LCBDs.

LCBD_df <- data.frame(
  "site" = rownames(species_mat_pa),
  "LCBD_repl" = LCBD.comp(beta[[1]] ^ 0.5 )[[2]],
  "LCBD_rich" = LCBD.comp(beta[[2]] ^ 0.5 )[[2]],
  "LCBD_D" = LCBD.comp(beta[[3]] ^ 0.5 )[[2]]  ) %>%
  column_to_rownames("site") %>%
  mutate(LCBD_repl = ifelse(LCBD_repl <0, abs(LCBD_repl), LCBD_repl))

The Ebro case study: LCBD
